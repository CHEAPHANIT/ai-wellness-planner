# NutriAI Submission Checklist

## Included

- Complete project source code
- All files required to run the application with Docker Compose
- README with project description, setup, technologies, run instructions, deployment link, and contributions
- 12-slide presentation in `.pptx` format
- Deployment/test link file
- Test credentials file

## Not Included

- Demo video, per current request

## Recommended Final Review

1. Update `Your Name` and `Your Course Name` on slide 1 if required.
2. Update the team member name in `README.md` if required.
3. Run `docker compose up --build -d`.
4. Open `http://localhost:8080`.
5. Register the test account from `Test_Credentials.txt` if the database is empty.
